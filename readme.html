Computer code for converting morphologies between hoc, swc, and
Knossos XML, and quantifying neuronal morphology as detailed in the
associated paper:

Otopalik AG, Goeritz ML, Sutton AC, Brookings T, Guerini CJ, Marder E
(2017) Sloppy morphological tuning in identified neurons of the
crustacean stomatogastric ganglion. Elife
http://dx.doi.org/10.7554/eLife.22352

are available at the author's github repository:

http://github.com/marderlab

See the readme and morphology.pdf in the repository for more information.
